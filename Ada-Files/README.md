# wb_ov5640_helper

This folder contains the Ada helper library for working with the Wishbone OV5640 camera controller.

Use this folder if you want to:
- Access the controller from Ada software
- Reuse helper code for camera setup and control
- Build software support for the OV5640 interface

# wb_ov5640_tests

This folder contains Ada test programs for the Wishbone OV5640 camera controller.

These tests are intended to validate the hardware interface and show how to use the helper library.

Use this folder if you want to:
- Run hardware validation tests
- See example usage from Ada
- Use the tests as a starting point for your own software
